subsistema de memória



Registrador

Armagena dados que em determinado tempo sera usado pelo processador.
O processador armagena e busca instruções em seu interios para que possa realizar as operações solicitadas ultilizando seus demais componentes.
memoria volatio.
muito limitada.




Cache

Fica entre a memoria principal (RAM) e o processador.
sua função principal é acelerar a velocidade de transferencia das informaçoes entre o processador e a RAM.

são volateis, e dependem de energia para manter seu conteudo

custo alto.


Memoria principal

É a memoria principal para o funcionamento
conhecida como memoria RAM
tem como finalidade manter armazenado os dados enquanto o computador estiver ligado.
volátil.

DDR - Double data rate (rua de tranferencia dupla)

HBM - Memoria em stack (empilhada) desenvolvida pela AMD, proximo do processador e mais rapida



Memoria ROM (Read only memory)
